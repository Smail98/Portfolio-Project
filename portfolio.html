I learned linux through this class and IT340 this semester. I discovered the origins of this operating system. I learned about basics commands, permissions, and user groups:

Commands

  • cd: allows for a change of directory. Its the same as changing folders on Windows.
  • ls: allows to view or "list" the content of the directory you're in.
  • find: is used to search for a specific file
  • cat: allows you to display the content of a file or multiple files
  • vi is the most popular text editor. It works across all platforms and distributions and is user-friendly. When entered with a filename, it pops up a new screen and has its own commands for editing.
